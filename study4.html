switch case

[多重選擇結構]

語法 :

switch (變數或運算式) {
    case 整數或字元 : //case數量可自行增加
        陳述句;
        break; //有沒有寫break會造成不同結果
    //default可以省略
    default : //沒有符合的整數或字元時執行
    //條件判斷為true進入此區塊
        陳述句;
        break; //已經是最底了，所以有沒有寫break都一樣
}
