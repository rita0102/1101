if、if...(else if)...else

if [單一選擇結構]

語法 :

if (條件式) {
//條件判斷為true進入此區塊
    陳述句;
}

若陳述句只有一行的話，
可以不加大括號

if...else [雙重選擇結構]

語法 :

if (條件式) {
//條件判斷為true進入此區塊
    陳述句;
}
else {
//條件判斷為false進入此區塊
    陳述句;
}

if...else if...else [多重選擇結構]

語法 :

if (條件式) {
//條件判斷為true進入此區塊
    陳述句;
}
else if (條件式) { //可以不只一個
//條件判斷為true進入此區塊
    陳述句;
}
else { //放在最後一個，可省略不寫
//以上條件皆不符合，進入此程式區塊
    陳述句;
}
